# PaleoFlow
### High-Fidelity Metabolic Tracking | Version 3.4

**PaleoFlow** is a minimalist, bio-hacking command center designed to track the metabolic transition from glucose-dependency to peak autophagy and cellular regeneration. Optimized for OLED displays, it provides a persistent, real-time visualization of your biological state across a 120-hour (5-day) spectrum.
