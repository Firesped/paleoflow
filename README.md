# PaleoFlow (MVP)

**PaleoFlow** is a minimalist metabolic state tracker designed to manage the "Wave" between feeding (**Build**) and fasting (**Flow**) windows. Built for precise metabolic control, it provides predictive modeling for ketosis milestones.

- **Integrated Biomarker Logs:** Manual entry and editing for Glucose (mg/dL) and Ketones (Blood, Breath, Urine).
- **Local-First Privacy:** Data resides exclusively on the user's device.
